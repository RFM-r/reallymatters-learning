# Самостоятельная работа по скрипту синхронизации репозиториев.

## Папка "AutoSync_bash" - Синхронизирует при помощи bash. Требуется использовать Git bash, чтобы запустить файлы.
## Папка "AutoSync_PowerShell" - Синхронизирует при помощи PowerShell. Запускать нужно Sync_run.bat(он служит таким некоторым "лаунчером")
